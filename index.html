My PortfolioMy Work

Here is some of my work that I've done in various programming languages.

WorldGame-Website-clone

Description:

This is a JavaScipt project website. It's a game where you have to guess the word that is being displayed on the screen. You have 4 chances and you lose one every time you guess a wrong letter. It generates new words every time you play and has hints and darkmode. This website was made using HTML, CSS and JavaScript.

NoteTakingApp-Website

Description:

This is a Note Taking App that I made using React and AWS for the backend. It allows you to create, edit, and delete notes. It also has a login and signup page that uses Google Oath to login.

Password-Generator-Website

Description:

This is a JavaScript project that generates a random password based on user-selected criteria. This app will run in the browser and feature dynamically updated HTML and CSS powered by JavaScript code. It also features a clean and it creates a password file that can be downloaded and saved on the user's computer for future use and the file can be used in the website for future modifications.

ZOO_Schedule_Builder

Description:

This is a JAVA project about building a schedule for a zoo. It is for the supervisor to build a schedule for the zookeepers and the animals. The supervisor can add, delete, and update the schedule and request additional volunteer if needed based on the scheduler and uses Swing for the GUI and MySQL for the database.

Museum-Project

Description:

This is a Python project that I made for a museum. It allows the user to create, edit, and delete exhibits. And also create user, edit user's privileges, and delete users. It uses command line to run the program and MySQL for the database.
